# FUTURE_DS_02
Data Science & Analytics Internship – Future Interns
📊TASk-2 Customer Retention & Churn Analysis
📌 Project Overview
Customer churn directly impacts revenue and long-term growth in subscription-based businesses.

This project analyzes customer subscription data to understand:

-Why customers leave

-Which customer segments are most at risk

-Customer lifetime patterns

-How businesses can improve retention

-The analysis reflects real-world work done by data analysts in SaaS and subscription companies.

🎯 Objective
1.The goal of this project is to:

2.Calculate overall churn rate

3.Identify high-risk customer segments

4.Analyze customer tenure and lifetime behavior

5.Provide actionable business recommendations

🛠 Tools & Technologies Used
Python

Pandas

NumPy

Matplotlib

Seaborn

Jupyter Notebook

📂 Dataset Information
Dataset Used: Telco Customer Churn Dataset (Kaggle)

🔎 Analysis Performed
1️⃣ Data Cleaning

2️⃣ Overall Churn Rate

3️⃣ Segment-Based Churn Analysis

4️⃣ Tenure & Customer Lifetime Analysis

5️⃣ Summary Export

📈 Key Insights
1.Customers with monthly contracts show significantly higher churn.

2.Most churn occurs within the first 12 months.

3.Higher monthly charges increase churn probability.

4.Customers using electronic check payment method show higher churn risk.

5.Long-term contract customers demonstrate strong retention behavior.
